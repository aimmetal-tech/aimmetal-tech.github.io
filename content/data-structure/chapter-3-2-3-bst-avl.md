## 二叉搜索树 BST

定义：左子树小于根，右子树大于根

**基本操作：插入、查找、删除**

## 平衡二叉树 AVL

| 失衡现象 | 旋转轴（中心） | 旋转方向     |
| -------- | -------------- | ------------ |
| LL       | 三点中心       | RR           |
| RR       | 三点中心       | LL           |
| LR       | 失衡点往下两点 | LR(先LL再RR) |
| RL       | 失衡点往下两点 | RL(先RR再LL) |
